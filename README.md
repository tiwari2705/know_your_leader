üöÄ Core Features
Secure User Authentication: Role-based access control for 'Citizens' and 'Admins' with secure, HTTP-only sessions.

Dynamic Leader Database: A comprehensive database of political leaders, searchable and filterable by name, party, and position.

Citizen Review System: Logged-in citizens can submit a 1-5 star rating and a detailed written review for any leader.

Aggregate Rating Engine: Leader profiles display an average rating and a detailed percentage breakdown of all submitted reviews.

AI Political Assistant: An integrated chatbot (powered by the Gemini API) that can answer general political questions using live Google Search data.

AI-Powered News Feed: The dashboard features a real-time "Trending Political News" section, generated by the Gemini API.

Detailed Leader Profiles: A modal-based deep-dive into each leader, including:

Biographical & Career Background

Financial Disclosures & Assets

Legal & Educational History

This project was built with security as a top priority, not an afterthought.

‚öôÔ∏è Installation & Setup
To run this project on your local machine, follow these steps:

Clone the Repository

Bash

git clone https://github.com/your-username/know-your-leader.git
cd know-your-leader
Database Setup

Create a new MySQL database (e.g., know_your_leader_db).

Import the know_your_leader_db.sql file (you will need to export this from your local phpMyAdmin) into your new database.

Configure Environment

Create a new file named .env in the root of the project.

Copy the following structure into it and add your local credentials:

Ini, TOML

# .env - Your local machine's secret keys
# (This file is ignored by Git)

# --- API KEYS ---
GEMINI_API_KEY="your-gemini-api-key-for-the-chatbot"
NEWS_API_KEY="your-gemini-api-key-for-the-news-feed"

# --- DATABASE ---
DB_HOST="localhost"
DB_USER="root"
DB_PASS="your-local-db-password"
DB_NAME="know_your_leader_db"
Run the Server

Place the entire project folder in your web server's root directory (e.g., htdocs for XAMPP).

Start your Apache and MySQL services.

Open your browser and navigate to http://localhost/know-your-leader.
